# e-plantShopping

**e-plantShopping** is a practice project that simulates a plant e-commerce store. The app lets users browse a plant catalog, view basic product details, and manage a shopping cart.

## Overview
This project was built for learning purposes to practice common web development concepts such as component-based UI, state management, dynamic rendering, and a basic shopping flow.

## Features
- Plant catalog (product listing)
- Product details (name, price, image, description)
- Shopping cart
  - Add items
  - Remove items
  - Update quantities
  - View order total

## Usage
- Browse the catalog and add plants to your cart.
- Adjust quantities and review the total before checkout.

## Project Status
Practice project in progress. Possible improvements include search, filters, a checkout flow, and cart persistence.
